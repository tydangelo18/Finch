# Finch marketing analyzer

## Description

- Allows a user to search for and save Tweets by keywords.

## Technologies Used

- React.JS
- MongoDB
- Node.js
- Express.js
- Material UI
- Twitter API and Documentation
- Postman
- Mongo Atlas
- Responsively App
- Concurrently
- Mongoose
- JSON Web Tokens
- Passport.js
- Google Fonts
- Medium
- Stack Overflow
- Heroku
