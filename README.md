# Finch Tweet Saver

## Description

- A simple tool used to search and save Tweets by keywords.

## Demo

- Deployed to [Heroku](https://agile-fjord-87962.herokuapp.com/).

## Technologies Used

- React.js
- MongoDB
- Node.js
- Express.js
- Material UI
- Twitter API
- Postman
- Mongoose
- Heroku
